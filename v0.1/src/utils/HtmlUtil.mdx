# HtmlUtil
HTML相关的辅助工具

### 富文本转换为纯文本
<Playground>
  <div>
    {
      HtmlUtil.plainHtml('<span>aaaa</span>')
    }
  </div>
</Playground>

```js
// 导出blob
HtmlUtil.exportBlob(blob, '文件名.txt');
```

### 复制到粘贴板
<Playground>
  <Button onClick={()=>{
    if(HtmlUtil.copyText('aaaa')){
      message.success('复制成功，可粘贴到其它地方');
    }
    else{
      message.error('复制失败');
    }
  }}>
    点我复制 aaaa
  </Button>
</Playground>
