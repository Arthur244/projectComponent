# LimitUpload
可设置最大上传文件数量的组件。

当设置type时，但未设置accept时，会自动设置accept

## 示例

### 默认，请观察开发者工具的输出，当文件上传成功、移除时，会触发onChange
<Playground>
  <LimitUpload 
    action="https://www.mocky.io/v2/5cc8019d300000980a055e76" 
    validateFile={(file)=>{
      if(file.response && file.response.status === 'done'){
        return true;
      }
      else{
        return false;
      }
    }}
    onChange={(fileList)=>{
      console.log('fileList', fileList);
    }}
  />
</Playground>

### 上传异常时

### 自定义上传异常的提示方法
<Playground>
  <LimitUpload

</Playground>

### 禁用
<Playground>
  <LimitUpload 
    disabled action="./" 
    validateFile={(file)=>{
      if(file.response && file.response.success){
        return true;
      }
      else{
        return false;
      }
    }} 
  />
  <hr />
  <LimitUpload disabled type={UploadType.IMAGE} action="./" />
</Playground>
