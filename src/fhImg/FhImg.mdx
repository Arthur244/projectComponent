# FhImg
简单的图片封装，支持三种不拉伸的图片填充模式，图片预览

### 主要做了以下工作

+ 屏蔽拉伸类不友好的图片填充模式
+ 图片地址及宽高用属性设置
+ 配置预览功能

### 后期
